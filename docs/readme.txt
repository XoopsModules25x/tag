Read Me First
=============

The tag module is designed for site-wide tag management, handling tag input,
display and stats for each module that enables tag plugin. This module provides
 a centralized toolkit including input, display, stats and substantial more
 comprehensive applications, so that each module does not need to develop its
 own tag handling scripts.

 Requirements
 _____________________________________________________________________

- PHP version >= 5.3.7
- XOOPS version >= 2.5.10
